# 47Chat - Lokalny Chatbot RAG

Inteligentny chatbot zdolny do prowadzenia rozmów na podstawie wgranych dokumentów PDF i Markdown, działający w 100% lokalnie.



---

## 🚀 Kluczowe Funkcje

* **Lokalne Przetwarzanie:** Wszystkie operacje, włączając w to modele AI, odbywają się na Twoim komputerze dzięki Ollama.
* **Obsługa Wielu Formatów:** Rozmawiaj z plikami `.pdf` i `.md`.
* **Intuicyjny Interfejs:** Prosty i przejrzysty interfejs czatu zbudowany w Streamlit.
* **Bezpieczeństwo i Prywatność:** Twoje dokumenty nigdy nie opuszczają Twojego komputera.

---

## 🛠️ Instalacja i Uruchomienie

Kroki potrzebne do uruchomienia projektu na nowym komputerze.

1.  **Sklonuj repozytorium:**
    ```bash
    git clone https://github.com/Hipson47/47chat.git
    cd 47chat
    ```

2.  **Zainstaluj Ollama:** Pobierz i zainstaluj [Ollama](https://ollama.com).

3.  **Pobierz model LLM:** Uruchom w terminalu komendę, aby pobrać domyślny model:
    ```bash
    ollama pull llama3
    ```

4.  **Uruchom aplikację:** Skorzystaj z dołączonego skryptu startowego, który automatycznie stworzy środowisko wirtualne i zainstaluje zależności:
    ```bash
    ./start_chatbot.bat
    ```
